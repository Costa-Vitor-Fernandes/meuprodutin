# MEUPRODUTIN 🛒

MEUPRODUTIN é uma plataforma de e-commerce focada na venda de produtos digitais (downloads e links de acesso), inspirada no Gumroad/Kiwify/Hotmart. Criadores podem criar suas lojas em subdomínios (ex: `loja.meuprodutin.com`) e vender diretamente para seu público de forma rápida e segura.

## 📦 Features Principais

- Lojas dinâmicas via subdomínios.
- Upload e entrega segura de arquivos digitais via *Presigned URLs* (s3).
- Checkout integrado suportando pagamentos únicos e assinaturas.
- Split automático de pagamentos via Stripe Connect (A plataforma cobra uma porcentagem por venda).
- Acesso à lista de compradores e e-mails pelo criador para campanhas de remarketing.
- Inserção de Pixels de rastreamento e ferramentas de Analytics na loja(?).
